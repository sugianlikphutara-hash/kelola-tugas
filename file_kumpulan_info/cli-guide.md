# 🚀 Panduan CLI untuk React + GitHub

## 📁 Navigasi Folder

``` bash
cd nama-folder
cd ..
dir
```

## ⚛️ React (Vite)

``` bash
npm install
npm run dev
npm run build
```

## 🧰 Git (Dasar)

``` bash
git init
git remote add origin https://github.com/username/repo.git
```

## 💾 Simpan Perubahan

``` bash
git add .
git commit -m "pesan perubahan"
```

## ⬆️ Upload ke GitHub

``` bash
git push
```

## ⬇️ Update dari GitHub

``` bash
git pull
```

## 🔍 Cek Status

``` bash
git status
```

## 🧹 Hapus File

``` bash
rm nama-file
```

## 🔁 Workflow Harian

``` bash
git add .
git commit -m "update fitur"
git push
```
